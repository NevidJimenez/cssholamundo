Selectores

La forma en la que invocamos los poderes de CSS en HTML es mediante los SELECTORES y hay 4 tipos de ellos:

  • Selector por etiqueta

    • Funciona escribiendo la etiqueta de html a la cual le queremos hacer magia con css.

    Ejemplo de selector por etiqueta

  • Selector por id

    • Este funciona dandole un id al elemento que lo necesitemos, pero es considerado una mala práctica ya que los id se suelen guardar para js, ademas de que los id son únicos así que solo afectaría 1 solo elemento.

    Ejemplo de selector por id

  • Selector por clase

    • Esta es la forma más utilizada para poner css ya que puede afectar 1 o varios elementos, ademas de que cada elemento html puede contener varias clases, solo separandolas por un espacio entre ellas.

    • ejemplo de clase 1
    • ejemplo de clase 2
    • ejemplo de clase 3

  • Selector universal

    • Se utiliza el asterisco * para el selector universal, aunque viene con un pequeño truco, éste solo afectará a los elementos que no hayan sido modificados por otro selector.

Formas de elegir el color

Existen 3 tipos de formas en las que podemos elegir los colores para css

  • hexadecimal

    • Después de la propiedad "color" ponemos el simbolo de # seguido de 3 o 6 caracteres que van desde el 0 a la f, siendo la f el valor máximo. Si solo utilizamos 3 caracteres css entendera que el segundo, el cuarto, y el sexto número será el mismo que su predecesor, por ejemplo, si ponemos #abc, se entenderá que quisiste decir #aabbcc, pero en su forma reducida.

  • rgb

    • Para esta forma colocamos "rgb()" y estos valores van desde 0 hasta 255, y significan: red, green, blue.

  • hsl

    • HSL significa: Hue Saturation Lightness y esta basado en una rueda de color, la saturación y el brillo que podemos darle a un color, midiendose como HSL(0 a 360, 0% a 100%, 0% a 100%) siendo un 0,100,50 un rojo brillante o 240, 100, 50 un azul muy intenso.

Unidades de medida

En css y en general se utilizan unidades de medida en pixeles ya que es la unidad mínima que se puede utilizar en una computadora y en la web. Algunas son:

  • pixel o px
  • centimetros o cm
  • pulgadas o in
  • milimetros o mm
  • puntos o pt
  • picas o pc
  • em o tamaño relativo actual del padre directo
  • rem o tamaño relativo actual del elemento relativo que sería html

Margin, Padding y border. Box model.

Margin lo que hace es que da un espacio desde el elemento hacia afuera y hasta el siguiente elemento.

Padding le da un espacio hacia adentro del elemento y hasta el siguiente elemento o contenido.

Border es la linea que rodea el elemento y también es manipulable como ya lo vimos.

Adicionando, existe otra propiedad llamada "outline" que es similar al "border" con la peculiaridad de que no genera tamaño en pixeles, sino que esta por encima de la pantalla, así que no afecta el diseño.
